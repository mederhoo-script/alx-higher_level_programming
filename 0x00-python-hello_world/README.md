## Usage
1. Execute the `0-run` script to run a Python script.
   ```bash
   ./0-run
   ```
2. Execute the `1-run_inline` script to run Python code directly.
   ```bash
   ./1-run_inline
   ```
3. Run the `2-print.py` Python script.
   ```bash
   python3 2-print.py
   ```
